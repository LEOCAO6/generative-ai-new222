## v1.0 Release
Llama 2 Chat Application

### Features
* Support for 4-bit and 8-bit quantization using accelerate and bitsandbytes.

## v0.9 Release
Llama 2 Chat Application

### Features
* Llama-2 chat application
    * Supported precision: BF16, FP32, FP16
    * Support for various Llama 2 models from HuggingFace
    * User friendly UI
